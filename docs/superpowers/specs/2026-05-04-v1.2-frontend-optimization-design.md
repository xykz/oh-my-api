# v1.2 前端界面优化 设计文档

## 目标

将 lingma2api Console 从"简陋单调"提升到生产级视觉体验。在保持现有 Glassmorphism CSS 设计系统的基础上，通过图标系统、加载状态、空态展示、布局重构和响应式适配完成全面升级。

## 范围

- **IN：** lucide-react 图标替换、Skeleton/Spinner/EmptyState 通用组件、Dashboard Grid 重构、Layout 导航修复、各页面加载态与空态、表格响应式
- **OUT（LATER）：** 实时动画图表、拖拽排序表格、深色模式独立配色重构、国际化 i18n

## 架构

保持现有 CSS 变量系统和 Glassmorphism 风格不变，新增 `lucide-react` 图标库，引入三个通用 UI 组件（Skeleton、Spinner、EmptyState），按需替换各页面的 emoji/文字按钮和简陋加载态。

## 新增通用组件

### Skeleton

- **用途：** 数据加载时的内容占位
- **变体：** `text`（单行文本）、`rect`（矩形卡片）、`circle`（圆形头像）
- **样式：** CSS shimmer 脉冲动画，背景色 `rgba(148,163,184,0.15)`，圆角继承系统变量
- **组合：** `SkeletonTable`（预设 6 行 x N 列的表格骨架）

### Spinner

- **用途：** 按钮操作、局部刷新的加载指示
- **尺寸：** `sm`(16px) / `md`(24px) / `lg`(32px)
- **样式：** CSS 旋转动画，边框使用 `--gradient-primary` 渐变色

### EmptyState

- **用途：** 无数据时的友好展示
- **结构：** `icon`（lucide 图标） + `title` + `description` + `action`（可选按钮）
- **变体：** `default`（仅文字图标）、`actionable`（带引导按钮）

## CSS 新增

在 `global.css` 追加：

- `@keyframes skeleton-shimmer` — 从左到右的光扫动画
- `.skeleton-*` 系列类 — 基础占位、文本、矩形、圆形
- `@keyframes spin` — Spinner 旋转
- `.spinner` 系列类 — 三种尺寸
- `.empty-state` — 居中布局、图标尺寸、文字颜色
- `.table-scroll` — 表格横向溢出容器
