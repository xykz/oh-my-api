First design decision

Which account selection model should the first implementation target?

A

One Global Pool

Every eligible China and International account participates in one load-balanced pool. Region adapter is chosen after account selection.

B

Region-Aware Pools

Accounts are grouped by region. Requests can use a default region, explicit region, or model-to-region mapping before balancing inside that group.

C

Primary With Fallback

Prefer one region or account set, then fall back to the other only on configured errors or quota exhaustion.
